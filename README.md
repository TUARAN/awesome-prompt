# Awesome Prompt - AI 提示词学习平台

专注于「技术写作上下文工程」。它不仅整理了从结构层到动态上下文管线的完整方法论，还提供可直接落地的模板、案例和实践建议。

## ✨ 项目亮点
- **上下文工程方法论**：以结构层 → 信息调度层 → 记忆层 → 动态上下文管线的四层模型组织全部内容。
- **丰富教学模块**：摘要生成、数据对比、角色模拟、创意写作、问题解决等教程可即学即用。
- **实战模板合集**：覆盖办公效率、内容创作、数据分析、编程开发等真实场景，并配备输入/输出示例。
- **沉浸式界面**：采用 Apple/Notion 风格设计、动态粒子背景、暗色模式和响应式布局，保证沉浸式体验。
- **一键操作**：内置提示词复制、分类筛选、场景跳转等交互细节，减少学习阻力。

## 🚀 快速开始
> Node.js ≥ 18.0.0、npm ≥ 9.0.0 推荐。

```bash
# 安装依赖
npm install

# 本地开发
npm run dev

# 生产构建
npm run build

# 本地预览构建结果
npm run preview
```

构建完成后，`dist/` 目录即为可部署的静态站点。

## 📚 核心学习内容
- **基础教程**：摘要生成、数据对比、角色模拟、创意写作、问题解决。
- **实用技巧**：明确性、上下文、约束性、迭代性的提示词写作准则。
- **应用场景**：办公效率、内容创作、数据分析、编程开发等案例库。
- **层级方法**：
  1. 结构层 —— 将写作要求、读者画像等信息结构化。
  2. 信息调度层 —— 不同写作阶段动态注入所需上下文。
  3. 记忆层 —— 保留多轮对话的上下文连续性。
  4. 动态上下文管线 —— 基于反馈持续优化的自适应系统。

## 🎨 设计与交互
- Apple/Notion 氛围的极简配色与排版。
- 动态粒子、渐变光晕、网格线等视觉装饰营造沉浸式体验。
- 深色模式根据 `prefers-color-scheme` 自动切换，可记忆用户选择。
- 固定侧边导航快速跳转四个学习阶段，组件提供一键复制提示词。

## 🧭 学习路径建议
1. **从基础原则开始**：确保明确性、上下文、约束性三大要素被表达清晰。
2. **结合真实场景**：选择与你工作最相关的案例进行实践。
3. **小步验证与复盘**：在多轮对话中迭代提示词，观察模型响应。
4. **沉淀模板**：将高质量提示词沉淀为结构化片段，便于复用。
